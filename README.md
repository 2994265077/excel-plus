# Feature

- Easy to use
- Annotation driven
- Based java 8
- Support `xls` and `xlsx`

# Usage

Add maven dependency

```xml
<dependency>
    <groupId>io.github.biezhi</groupId>
    <artifactId>excel-plus</artifactId>
    <version>0.0.1</version>
</dependency>
```
